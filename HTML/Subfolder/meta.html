Introducing metadata

Meta is placed in the head tag

Attribute Description Example
charset Specifies the character encoding for the HTML document. <meta charset="utf-8">
http-equiv Provides an HTTP header for the information/value of the content attribute. <meta http-equiv="refresh content="3">
name Specifies a name for the metadata. e.g. author, description, viewport, ... etc. <meta name="author" content="James">
