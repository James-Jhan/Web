Introducing List Symbol

It's placed in the body tag

  • To define the list number

    • Example 1:

    code:

    <ol start="1" type="1">
        <li>No.1</li>
        <li>No.2</li>
        <li>No.3</li>
    </ol>

    result:

    1. No.1
    2. No.2
    3. No.3

    Example 2:

    code:

    <ol start="2" type="A">
        <li>No.1</li>
        <li>No.2</li>
        <li>No.3</li>
    </ol>

    result:

    1. No.1
    2. No.2
    3. No.3
  • To defines an unordered list

    • Example 1:

    code:

    <ul style="list-style: disc">
        <li>No.1</li>
        <li>No.2</li>
        <li>No.3</li>
    </ul>

    result:

    • No.1
    • No.2
    • No.3

    Example 2:

    code:

    <ul style="list-style: square">
        <li>No.1</li>
        <li>No.2</li>
        <li>No.3</li>
    </ul>

    result:

    • No.1
    • No.2
    • No.3
