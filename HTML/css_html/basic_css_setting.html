Font style

  • Font size.

    • The font size can be decided by following methods:

    Method 1
    Unit Description
    px(pixel) Default value.
    pt(point) It's usually used on printing. 1 pt = 1/72 inch.
    pc(picas) It's usually used on printing. 1 pc = 12 pts.
    in(inch) 1 in = 2.54 cm.
    cm centimeter.
    mm millimeter.
    em Inherit font size from upper layer.
    ex The reference is according to height of lowercase.
    Method 2
    Property Description
    xx-small x-small is equal to 120% xx-small.
    x-small small is equal to 120% x-small.
    small medium is equal to 120% small.
    medium It's the basis of font size by using method 2.
    large large is equal to 120% medium.
    x-large x-large is equal to 120% large.
    xx-large xx-large is equal to 120% x-large.

    Example:
    font-size: 18px;
     font-size: 18pt;
     font-size: 2cm;
     font-size: 1.8em;
     font-size: medium;
     font-size: x-small;
     font-size: x-large;
     font-size: 200%;
     font-size: x-small;
     font-size: x-large;
