Introducing Paragraph Attribute

Paragraph setting is placed in the body tag

  • pre:Preformatted text.

    • code:

    <pre>
                日照香爐生紫煙，遙看瀑布掛前川。<br/>
    
                飛流直下三千尺，疑是銀河落九天。
    </pre>

    result:
                日照香爐生紫煙，遙看瀑布掛前川。
                飛流直下三千尺，疑是銀河落九天。
  • blockquote:A section that is quoted from another source.

    • code:

    <blockquote>
    日照香爐生紫煙，遙看瀑布掛前川。<br/>
    飛流直下三千尺，疑是銀河落九天。
    </blockquote>

    result:
    日照香爐生紫煙，遙看瀑布掛前川。
    飛流直下三千尺，疑是銀河落九天。
  • h1~h6:The six different HTML headings.

    • code:

    <h1>This is heading 1</h1>
    <h2>This is heading 2</h2>
    <h3>This is heading 3</h3>
    <h6>This is heading 6</h6>
    <h4>This is heading 4</h4>
    <h5>This is heading 5</h5>
    result:

    This is heading 1

    This is heading 2

    This is heading 3

    This is heading 4

    This is heading 5
    This is heading 6
  • p:A paragraph is marked up as follows.

    • code:

    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
    <p>This is a paragraph</p>
    result:

    This is a paragraph

    This is a paragraph

    This is a paragraph
