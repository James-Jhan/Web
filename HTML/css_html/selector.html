  • Attribute Selectors

Description:

It's usually designed to the hyperlink, we designate the styles of them by this selector.

Form Description Code Example
Tag[Property] The Property will be selected whcih the same tag. a[href]{
color: red;
} 		Google website
Tag[Property = "Value"] The property will be selected which tag and value are same. a[href = "https://www.google.com/"]{
font-size: 20px;
color: red;
} 		Google website
Tag[Property ~= "Value"] The property whcih content has assigned value will be selected. a[href ~= "https://www.google.com.tw/"]{
font-size: 10px; color: orange; } 		Taiwan Google website
Tag[Property |= "Value"] The property whcih start value and "-" symbol is assigned value will be selected. img[alt |= FireFox]{
border:2px solid blue;
}
Tag[Property ^= "Value"] The property which started words as same as assigned value. img[alt ^= "Ame"]{
boreder: 2px solid green;
}
Tag[Property $= "Value"] The property which ended words as same as assigned value will be selected. img[alt $= "land"]{
border: 2px solid red; }
Tag[Property *= "Value"] The property which has any one assigned words will be selected. img[alt *= "ML"]{
color: white;
background-color: black;
}
