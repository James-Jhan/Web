    Key word Description
    <img> To define an image.
    src To define the URL of the image.
    alt To define an alternate text for an image, if it cannot be displayed.
    width and height To define the size of the image.
    <map> To define an image-map.
    <area> To define clickable areas in the image-map.
    <img>'s element usemap To point to an image-map.
    <picture> To show different images for different devices.

  • Loading Audio

    • code:
    <audio controls>
        <source src="mp3 source" type="audio/mpeg"/>
        <source src="ogg source" type="audio/ogg"/>
        <Some Comment(It's will be showed when audio file is not exist.)>
    </audio>

    result:

    Key word Description
    autoplay Specifies that the audio will start playing as soon as it is ready.
    controls Specifies that audio controls should be displayed (such as a play/pause button etc).
    loop Specifies that the audio will start over again, every time it is finished.
    muted Specifies that the audio output should be muted.
    preload Specifies if and how the author thinks the audio should be loaded when the page loads.
    src Specifies the URL of the audio file.

  • Loading Video

    • Example 1:Loading video by files

    code:
    <video width="450" height="300" controls>
        <source src="video_source.mp4" type="video/mp4"/>
        <source src="video_source.ogg" type="video/ogg"/>
    </video>

    result:

    video attribute list:

    Key word Description
    width and height To define the size of the video.
    controls Specifies that video controls should be displayed (such as a play/pause button etc).
    autoplay Specifies that the video will start playing as soon as it is ready
    loop Specifies that the video will start over again, every time it is finished

    Example 2:Loading video by youtube

    code:
    <iframe width="650" height="500" src="https://www.youtube.com/embed/youtube_ID?autoplay=1" frameborder="0"/>
    result:

    youtube video attribute list:

    Key word Description Example
    width and height To define the size of the video. <iframe width="width" height="height">
    controls Specifies that video controls should be displayed (such as a play/pause button etc). <iframe src="https://www.youtube.com/embed/youtube_ID?controls=0">
    autoplay Specifies that the video will start playing as soon as it is ready <iframe src="https://www.youtube.com/embed/youtube_ID?autoplay=1">
    loop Specifies that the video will start over again, every time it is finished <iframe src="https://www.youtube.com/embed/youtube_ID?playlist=youtube_ID&loop=number">
